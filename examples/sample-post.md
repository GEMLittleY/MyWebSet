# 我的免费博客搭建记录

今天终于把个人博客搭建完成了！全程零花费，记录一下过程。

## 使用的技术栈

- **Next.js** - 前端框架，自带服务端渲染
- **Supabase** - 免费数据库，存储文章数据
- **Vercel** - 自动部署平台，push 代码就上线
- **Cloudflare** - CDN 加速和安全防护

## 为什么选择这套方案？

1. 完全免费，适合个人项目
2. 性能极佳，全球 CDN 加速
3. 开发体验好，git push 即部署
4. 安全可靠，有企业级 DDoS 防护

## 总结

这套方案非常适合想要搭建个人博客的开发者，推荐大家试试！
